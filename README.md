# Finite State Machine (FSM) Explorer
 A Finite State Machine exploration tool for engineering education. Written in Javascript and HTML with no dependencies.
## Try it Out
[Click here](https://dougsummerville.github.io/FSM-Explorer/fsmexplorer.html) to try it out live.
## About
Finite State Machine (FSM) Explorer is an educational tool for exploring FSM controllers, the type of FSM used by engineers when designing digital systems. The tool allows the user to draw an FSM diagram, specify node and arc parameters, and functionally simulate the resulting FSM.
## Features
### FSM Editor
* Unlimited history with undo/redo
* File save and load
* Export FSM image
* Autoarrange to help create nice looking diagrams for export
* Automatic or manual state and input/output signal naming 
<* Support for Mealy and Moore outputs 
* Support for several styles of specifying arc inputs and FSM outputs 

### FSM Simulator

* Accurate functional simulation including multiply-driven (X) and unknown (U) outputs 
* Export timing diagram as image 
* Timing pointer to help read diagram 
* Timing pointer snap to clock edges with before and after active edge snapping 
* Graphical input waveform editor 

## Getting Started
The easiest way to get started is to click Help and select one of the example FSMs to play around with.  The Help section will also provide some information on using the drawing and simulation tools.
